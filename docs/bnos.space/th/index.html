คำถามที่พบบ่อย (FAQ)

คำถามทั่วไปเกี่ยวกับ BnOS

BnOS คืออะไร?

BnOS (Bitcoin Nostr Operating System) คือระบบจุดขาย (POS) และระบบปฏิบัติการธุรกิจแบบกระจายอำนาจที่ขับเคลื่อนด้วย Lightning Network ช่วยให้ธุรกิจรับการชำระเงินด้วย Bitcoin และจัดการการดำเนินงานได้โดยไม่ต้องผ่านคนกลาง

BnOS ใช้งานฟรีหรือไม่?

ใช่ BnOS เป็นซอฟต์แวร์โอเพ่นซอร์ส คุณสามารถใช้คุณสมบัติพื้นฐานได้ฟรี เราไม่เรียกเก็บค่าธรรมเนียมการทำธุรกรรมจากการชำระเงินของคุณ

ฉันจำเป็นต้องมีบัญชีธนาคารหรือไม่?

ไม่ คุณไม่จำเป็นต้องมีบัญชีธนาคารแบบเดิม BnOS ทำงานบน Bitcoin Lightning Network ช่วยให้คุณควบคุมเงินของคุณได้อย่างเต็มที่

โหมดออฟไลน์ทำงานอย่างไร?

BnOS ใช้สถาปัตยกรรม 'Local-First' ข้อมูลจะถูกเก็บไว้ในอุปกรณ์ของคุณก่อน คุณสามารถดำเนินการขายแบบออฟไลน์ได้ และข้อมูลจะถูกซิงค์กับ Nostr relays เมื่อคุณเชื่อมต่อออนไลน์อีกครั้ง

พร้อมที่จะเริ่มต้นแล้วหรือไม่?

เข้าร่วมรายชื่อรอ และ เป็นคนแรกที่ได้สัมผัสกับอนาคตของการดำเนินธุรกิจ
